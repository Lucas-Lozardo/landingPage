Backend Developer | Java & Spring Boot

Sou desenvolvedor backend com 3 anos de experiência em Java. Tenho foco na criação de APIs RESTful utilizando o framework Spring Boot, com persistência de dados em bancos relacionais como PostgreSQL e H2 DataBase. Tenho familiaridade com boas práticas de desenvolvimento, como princípios SOLID, testes automatizados com JUnit e versionamento de código com Git. Atualmente, estou ampliando meus conhecimentos para além do backend, estudando HTML e CSS com o objetivo de entender melhor o funcionamento da camada de apresentação e como ela se comunica com as APIs que desenvolvo. Essa jornada está me ajudando a construir projetos completos, mesmo que simples, e aprimorar minha visão de desenvolvimento web como um todo.
